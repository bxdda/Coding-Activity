# OpenWeather Application
1. Weather Application that uses a WeatherStation & GeoStation to get weather from geocode.
2. Takes user's city input, uses GeoStation to find geoData, uses WeatherStation to find weatherData on given geoData.
3. Allows favourites to be stored and edited.

## HOW TO RUN 
1. Download / Pull files. Extract if zipped.
2. run javac -d out src\openWeather\*.java src\module-info.java
3. run java -cp out openWeather.WeatherApplication
